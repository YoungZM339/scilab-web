# 社区行为准则

## 我们的承诺

我们承诺为所有参与者提供友善、安全、尊重且无骚扰的协作环境，不因年龄、身体、民族、性别认同、经验水平、国籍、外貌、宗教、性取向或其他个人特征区别对待。

## 期望行为

- 尊重不同观点，围绕事实、代码和用户需求讨论。
- 提供可执行、建设性的反馈，主动承认并修正错误。
- 保护用户、研究人员和贡献者的隐私与安全信息。
- 在分歧中保持克制，把社区整体利益置于个人输赢之上。

## 不可接受行为

- 侮辱、威胁、歧视、骚扰、跟踪或发布他人私人信息；
- 带有性暗示的言论或图像，以及不受欢迎的关注；
- 恶意破坏讨论、代码、数据或基础设施；
- 在维护者要求停止后继续不当行为。

## 执行

请通过仓库维护者提供的私密渠道报告问题。维护者会保护报告者隐私，并可根据严重程度采取提醒、暂时限制、永久移除等措施。维护者自身也受本准则约束。

本准则适用于项目仓库、Issue、Pull Request、讨论区以及代表项目参与的其他公共空间。
